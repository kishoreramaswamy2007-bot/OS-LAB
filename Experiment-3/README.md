# Experiment 3 - System Calls

## Aim

To study and implement basic UNIX system calls using C programming and Shell scripting.

## Objectives

- Understand process creation using `fork()`.
- Learn process identification using `getpid()`.
- Study process synchronization using `wait()`.
- Understand process termination using `exit()`.
- Learn file descriptor management using `close()`.

## Programs Included

1. Process Creation using `fork()`
2. Process Synchronization using `wait()`
3. File Descriptor Management using `close()`

## Software Requirements

- Kali Linux
- GCC Compiler
- Bash Shell

## Compile and Run

Compile the C program:

```bash
gcc filename.c -o output
./output
```

Run the shell script:

```bash
chmod +x filename.sh
./filename.sh
```

## Result

Thus, the UNIX system calls were studied and implemented successfully.
