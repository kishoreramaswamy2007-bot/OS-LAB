# Experiment 2 - UNIX Commands and Shell Programming

## Aim

To study and execute UNIX commands and implement shell programming using Bash in the Linux environment.

## Objectives

- Study basic UNIX commands.
- Learn file and directory management.
- Understand shell scripting fundamentals.
- Implement programs using loops and conditional statements.
- Execute shell scripts in the Linux environment.

## How to Execute

Make the script executable:

```bash
chmod +x filename.sh
```

Run the script:

```bash
./filename.sh
```

or

```bash
bash filename.sh
```
