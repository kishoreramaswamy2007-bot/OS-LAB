# Experiment 4 - CPU Scheduling Algorithms

## Aim

To implement various CPU Scheduling Algorithms using C programming and Shell scripting.

## Objectives

- Study First Come First Serve (FCFS) Scheduling.
- Study Shortest Job First (SJF) Scheduling.
- Study Priority Scheduling.
- Study Round Robin Scheduling.
- Calculate Waiting Time and Turnaround Time for each algorithm.

## Algorithms Implemented

1. First Come First Serve (FCFS)
2. Shortest Job First (SJF)
3. Priority Scheduling
4. Round Robin Scheduling

## Software Requirements

- Kali Linux
- GCC Compiler
- Bash Shell

## Compile and Run

Compile the C program:

```bash
gcc filename.c -o output
./output
```

Run the shell script:

```bash
chmod +x filename.sh
./filename.sh
```

## Result

Thus, the CPU Scheduling Algorithms were implemented successfully, and the waiting time and turnaround time were calculated.
